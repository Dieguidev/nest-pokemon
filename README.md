# Description

API Pokemon

## 1. Installation

```bash
$ npm install
```
## 2. Tener Nest CLI instalado
```bash
$ npm i -g @nestjs/cli
```
## 3. Levantar la base de datos
```bash
$ docker-compose up -d
```
## 4. Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
